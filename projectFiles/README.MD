# Python Quiz Game 🐍

A simple command-line Python Quiz Game that tests basic Python programming knowledge through multiple-choice questions.

## Features

* 5 Python-related quiz questions
* Multiple-choice answers (A/B/C/D)
* Instant feedback for each question
* Final score calculation
* Percentage display
* Performance feedback based on score

## Technologies Used

* Python 3

## How to Run

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder:

```bash
cd QuizProject
```

3. Run the program:

```bash
python quiz_game.py
```
